Introduction to C

C is a general-purpose programming language developed in the early 1970s. It is widely used for system programming, game development, and embedded systems. C provides a foundation for many modern languages like C++, Java, and Python.

Basic Structure of a C Program

#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

Key Concepts for Beginners

1. Variables
Used to store data. Example:
int age = 20;
float salary = 5000.50;
char grade = 'A';
2. Data Types
Common types:
int      // integer numbers
float    // decimal numbers
char     // single character
double   // larger decimal numbers
3. Operators
Examples:
+   -   *   /   %   // arithmetic
==  !=  <  >  <= >= // comparison
&&  ||  !           // logical
4. Loops
Repeated execution:
for(int i=0; i<5; i++) { ... }
while(condition) { ... }
do { ... } while(condition);
5. Conditional Statements
Example:
if(score >= 50) {
    printf("Pass");
} else {
    printf("Fail");
}
6. Functions
Used to organize code:
int sum(int a, int b) {
    return a + b;
}

Example Program: Simple Calculator

#include <stdio.h>

int main() {
    int a, b;
    char op;
    printf("Enter first number: ");
    scanf("%d", &a);
    printf("Enter operator (+, -, *, /): ");
    scanf(" %c", &op);
    printf("Enter second number: ");
    scanf("%d", &b);

    if(op == '+') printf("%d + %d = %d", a, b, a+b);
    else if(op == '-') printf("%d - %d = %d", a, b, a-b);
    else if(op == '*') printf("%d * %d = %d", a, b, a*b);
    else if(op == '/') printf("%d / %d = %d", a, b, a/b);
    else printf("Invalid operator!");
    
    return 0;
}
