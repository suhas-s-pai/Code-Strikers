Introduction to Fortran

Fortran (Formula Translation) is one of the oldest programming languages, mainly used for scientific computing, engineering, and numerical analysis.

Basic Structure of a Fortran Program

PROGRAM HELLO
    PRINT *, "Hello, World!"
END PROGRAM HELLO

Key Concepts for Beginners

1. Variables & Data Types
INTEGER :: age
REAL :: salary
CHARACTER(len=20) :: name
LOGICAL :: isStudent
age = 20
salary = 5000.50
name = "Suhas"
isStudent = .TRUE.
2. Arithmetic Operations
sum = a + b
difference = a - b
product = a * b
quotient = a / b
3. Conditional Statements
IF (score >= 50) THEN
    PRINT *, "Pass"
ELSE
    PRINT *, "Fail"
END IF
4. Loops
DO i = 1, 5
    PRINT *, i
END DO
5. Functions / Subroutines
FUNCTION sum(a,b)
    INTEGER :: a,b,sum
    sum = a + b
END FUNCTION sum
6. Input & Output
PRINT *, "Enter your age:"
READ *, age
PRINT *, "Your age is ", age

Example Program: Simple Addition

PROGRAM ADDITION
    INTEGER :: num1, num2, sum
    num1 = 10
    num2 = 20
    sum = num1 + num2
    PRINT *, "SUM = ", sum
END PROGRAM ADDITION
