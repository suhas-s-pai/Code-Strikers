Introduction to C++

C++ is an extension of C that includes object-oriented programming features. It is widely used for system/software development, games, and high-performance applications.

Basic Structure of a C++ Program

#include <iostream>
using namespace std;

}

Key Concepts for Beginners

1. Variables & Data Types
int age = 20;
float salary = 5000.50;
char grade = 'A';
string name = "Suhas";
2. Operators
+ - * / %        // arithmetic
== != < > <= >=  // comparison
&& || !          // logical
3. Loops
for(int i=0;i<5;i++){...}
while(condition){...}
do{...} while(condition);
4. Conditional Statements

    cout << "Pass";
}else{
    cout << "Fail";
}
5. Functions

Example Program: Simple Calculator

#include <iostream>
using namespace std;

int main(){
    int a,b;
    char op;
    cout << "Enter first number: "; cin >> a;
    cout << "Enter operator (+,-,*,/): "; cin >> op;
    cout << "Enter second number: "; cin >> b;

    if(op=='+') cout << a+b;
    else if(op=='-') cout << a-b;
    else if(op=='*') cout << a*b;
    else if(op=='/') cout << a/b;
    else cout << "Invalid operator!";
}
