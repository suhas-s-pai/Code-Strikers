Key Concepts for Beginners

1. Variables & Data Types
age = 20
salary = 5000.50
grade = 'A'
name = "Suhas"
is_student = True
2. Operators
+ - * / %        # arithmetic
== != < > <= >=  # comparison
and or not       # logical
3. Loops
for i in range(5):
    print(i)

while condition:
    # do something
4. Conditional Statements
if score >= 50:
    print("Pass")
else:
    print("Fail")
5. Functions
def sum(a, b):
    return a + b
6. Lists & Dictionaries
fruits = ["apple", "banana", "cherry"]

Example Program: Simple Calculator

a = int(input("Enter first number: "))
op = input("Enter operator (+, -, *, /): ")
b = int(input("Enter second number: "))

if op == '+':
    print(a + b)
elif op == '-':
    print(a - b)
elif op == '*':
    print(a * b)
elif op == '/':
    print(a / b)
else:
    print("Invalid operator!")
