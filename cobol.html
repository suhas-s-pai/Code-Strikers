Introduction to COBOL

COBOL (Common Business Oriented Language) is one of the oldest programming languages. It is mainly used for business, finance, and administrative systems for companies and governments.

Basic Structure of a COBOL Program

IDENTIFICATION DIVISION.
PROGRAM-ID. HELLO-WORLD.

PROCEDURE DIVISION.
    DISPLAY "Hello, World!".
    STOP RUN.

Key Concepts for Beginners

1. Variables & Data Types
01 AGE        PIC 9(2) VALUE 20.
01 SALARY     PIC 9(5)V99 VALUE 5000.50.
01 NAME       PIC A(20) VALUE "Suhas".
2. Arithmetic Operations
COMPUTE TOTAL = PRICE + TAX
COMPUTE DIFFERENCE = A - B
3. Conditional Statements
IF SCORE >= 50
    DISPLAY "Pass"
ELSE
    DISPLAY "Fail"
END-IF
4. Loops
PERFORM VARYING I FROM 1 BY 1 UNTIL I > 5
    DISPLAY I
END-PERFORM
5. Sections & Paragraphs
PROCEDURE DIVISION.
    PARAGRAPH1.
        DISPLAY "Welcome".
    PARAGRAPH2.
        DISPLAY "COBOL Basics".
6. Display & Input
DISPLAY "Enter your age:".
ACCEPT AGE.
DISPLAY "Your age is " AGE.

Example Program: Simple Addition

IDENTIFICATION DIVISION.
PROGRAM-ID. ADDITION.

DATA DIVISION.
WORKING-STORAGE SECTION.
01 NUM1 PIC 9(2) VALUE 10.
01 NUM2 PIC 9(2) VALUE 20.
01 SUM  PIC 9(3).

PROCEDURE DIVISION.
    COMPUTE SUM = NUM1 + NUM2
    DISPLAY "SUM = " SUM
    STOP RUN.
